## Project Overview

The **Engineering Research Engine** allows researchers to browse and search for articles, submit their own papers (with permission), and interact with a professional research management system. Admins can manage articles and users via a dedicated dashboard.

---

## Key Features

- Secure **Sign Up / Login system** with role-based access
  - **Admin Dashboard:** control article publishing and user management
  - **User Accounts:** researchers can browse and access published articles
- Centralized **MS SQL Server database**
- Search functionality by:
  - Engineering category
  - Research title
- Multi-language support (English, French, and others)
- Authors can request permission to publish their papers
- Random article suggestion feature
